* 🔗 Professional URL Shortener

A modern and professional URL Shortener web application built using **Node.js**, **Express.js**, **MongoDB Atlas**, and **Vanilla JavaScript**.

It allows users to convert long URLs into short, shareable links with instant redirection.

## ✨ Features

* 🔗 Shorten Long URLs
* 🚀 Instant Redirection
* 💾 MongoDB Atlas Database
* 📋 Copy Short URL
* 🌐 Open Original URL
* 🎨 Modern Glassmorphism UI
* 📱 Responsive Design
* ⚡ Fast Express Backend
* ☁️ Deployed on Render
